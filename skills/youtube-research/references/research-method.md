# YouTube research method

## Source matrix fields

| Field | Purpose |
|---|---|
| Video ID and URL | Stable provenance |
| Channel and date | Source context |
| Inclusion reason | Selection transparency |
| Main thesis | Per-source position |
| Key claims | Comparable assertions |
| Evidence type | Data, demo, anecdote, citation, opinion |
| Timestamps | Auditable support |
| Caveats | Limits stated by the source |
| Confidence | Researcher's grounded assessment |

## Synthesis labels

- **Consensus:** multiple independent sources support the same finding.
- **Recurring claim:** repeated but not independently verified.
- **Disagreement:** sources make incompatible claims.
- **Gap:** the selected corpus does not answer the question.
- **Inference:** analyst conclusion derived from cited evidence.

## Minimum quality check

- Confirm every major finding maps to at least one source and timestamp.
- Prefer two independent sources for cross-video conclusions.
- Verify numbers and named entities when auto-captions may be wrong.
- State how sources were selected and what may be missing.
